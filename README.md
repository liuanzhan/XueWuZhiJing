## ✨ 项目特色

### 🎯 核心定位
一个专注于**鸿蒙生态系统**和**仓颉编程语言**的技术学习平台，为开发者提供从入门到精通的完整学习路径。

### 📚 丰富内容
- **鸿蒙开发教程**：从零开始的HarmonyOS应用开发全攻略
- **仓颉语言指南**：全新编程语言的系统化学习资料
- **工程化实践**：软件开发最佳实践与开发技巧分享
- **资源整合**：精选开源项目、开发工具与学习资料

### 🛠️ 技术栈
- **前端框架**：VuePress 2.x + Plume主题
- **构建工具**：Vite + TypeScript
- **内容系统**：Markdown + 丰富的扩展语法
- **部署平台**：支持多平台部署与CDN加速

## 🚀 快速开始

### 本地开发
```bash
# 克隆项目
git clone https://gitcode.com/liuanzhan/XueWuZhiJing.git

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 内容创作
项目采用标准的Markdown格式，支持丰富的扩展语法：
- 📝 **代码演示**：支持Python等语言的在线运行
- 🎬 **多媒体嵌入**：B站、YouTube视频无缝集成
- 📊 **数据可视化**：图表、流程图等富媒体内容
- 🎨 **样式增强**：自定义CSS与主题配置
